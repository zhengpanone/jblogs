标识符
=========================

指在程序中,我们自定义的内容。如类名、方法名、变量等都是标识符

命名规则
>>>>>>>>>>>>>>>>>>>>>>

- 标识符可以包含字母a-zA-Z、数字0-9、$、_等
- 标识符不能以数字开头
- 标识符不能是关键字

命名规范
>>>>>>>>>>>>>>>>>>>>>>>>

- 类名:首字母大写,后面每个单词首字母大写(大驼峰)
- 变量: 首字母小写,后面每个单词首字母大写(小驼峰)
- 方法: 同变量

数据类型
============================

基本数据类型
>>>>>>>>>>>>>>>>>>>>>

- 整数型 

    - byte 一个字节 取值范围 -128~127 
    - short 两个字节 -32768~32767  
    - int 4个字节-2^31 ~2^31-1 
    - long 8个字节 -2^63~2^63-1

- 浮点型 

    - float 单精度浮点数 4个字节 
    - double 双精度浮点数 8个字节

- 字符型 

    - char 两个字节 0-65535

- 布尔型 

    - boolean 一个字节 true false 

注意事项

- 字符串不是基本类型,而是引用类型

- 浮点型可能只是一个近似值,并非精确值

- 数据范围与字节数比一定相关,如float数据范围比long更加广泛,但是float是4字节,long是8字节

- 浮点数中默认类型是double,要使用float类型需要加上一个后缀F

- 整数默认类型为int,使用long类型需要加上一个后缀L
